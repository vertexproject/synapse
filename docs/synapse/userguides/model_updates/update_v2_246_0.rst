######################
v2.246.0 Model Updates
######################

The following model updates were made during the ``v2.246.0`` Synapse release.

**************
New Properties
**************

``it:exec:url``
  The form had the following properties added to it:


  ``page:components``
    The software components included in the rendered page.


  ``page:favicon``
    The favicon of the rendered page.


  ``page:title``
    The title of the rendered page.
