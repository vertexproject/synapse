######################
v2.233.0 Model Updates
######################

The following model updates were made during the ``v2.233.0`` Synapse release.

*********
New Forms
*********

``it:dev:function:sample``
  An instance of a function in an executable.


``it:dev:function``
  A function defined by code.



**************
New Properties
**************

``proj:project``
  The form had the following property added to it:

  ``platform``
    The platform where the project is hosted.
