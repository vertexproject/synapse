v2.231.0 Model Updates
######################

The following model updates were made during the ``v2.231.0`` Synapse release.

*********
New Forms
*********

``file:mime:pdf``
  Metadata extracted from a Portable Document Format (PDF) file.



**************
New Properties
**************

``risk:compromise``
  The form had the following property added to it:

    A tag used to associate nodes with the compromise.
