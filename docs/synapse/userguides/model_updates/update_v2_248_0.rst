######################
v2.248.0 Model Updates
######################

The following model updates were made during the ``v2.248.0`` Synapse release.

*********
New Forms
*********

``it:os:posix:cron``
  A cron job entry configured on a host.



**************
New Properties
**************

``inet:http:request``
  The form had the following property added to it:

  ``fetch``
    The fetch or page load event which caused the HTTP request.
