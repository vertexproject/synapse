######################
v2.150.0 Model Updates
######################

The following model updates were made during the ``v2.150.0`` Synapse release.

*************
Updated Types
*************

``inet:url``
  The ``inet:url`` type now recognizes UNC network paths and converts
  them into ``smb://`` URLs.
