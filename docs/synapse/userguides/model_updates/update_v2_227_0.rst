######################
v2.227.0 Model Updates
######################

The following model updates were made during the ``v2.227.0`` Synapse release.

*********
New Forms
*********

``ou:candidate:referral``
  A candidate being referred by a contact.



**************
New Properties
**************

``meta:feed``
  The form had the following property added to it:

  ``id``
    An identifier for the feed.


``ou:technique``
  The form had the following property added to it:

  ``names``
    An array of alternate names for the technique.


``ps:workhist``
  The form had the following property added to it:

  ``desc``
    A description of the work done as part of the job.


``risk:mitigation``
  The form had the following property added to it:

  ``names``
    An array of alternate names for the mitigation.


``transport:sea:vessel``
  The form had the following property added to it:

  ``callsign``
    The callsign of the vessel.
