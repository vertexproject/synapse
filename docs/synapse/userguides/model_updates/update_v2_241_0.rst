######################
v2.241.0 Model Updates
######################

The following model updates were made during the ``v2.241.0`` Synapse release.

*********
New Forms
*********

``meta:cluster:type:taxonomy``
  A type taxonomy for ``meta:cluster`` nodes.


``meta:cluster``
  A cluster of analytically relevant nodes generated by a specific source.



**************
New Properties
**************

``crypto:currency:block``
  The form had the following property added to it:

  ``chain``
    The chain where the block is recorded.


``crypto:currency:transaction``
  The form had the following property added to it:

  ``block:chain``
    The chain where the transaction is recorded.


``it:prod:soft``
  The form had the following property added to it:

  ``tag``
    The tag used to annotate nodes that are associated with the software.



***********
Light Edges
***********

``uses``
    When used with a ``risk:threat`` and an ``inet:service:platform`` node, the
    edge indicates the threat cluster uses the service platform.



****************
Deprecated Edges
****************

``uses``
    The edge has been deprecated when used with a ``risk:threat`` and an
    ``inet:service:app`` node. Please use the ``inet:service:platform``
    form instead.
